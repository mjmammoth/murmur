# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| Latest release | Yes |
| Older releases | No |

## Reporting a Vulnerability

Please report vulnerabilities privately through GitHub Security Advisories:

- https://github.com/mjmammoth/murmur/security/advisories/new

Do not open public GitHub issues for security vulnerabilities.

Include as much of the following as possible:

- A clear description of the vulnerability and potential impact
- Reproduction steps or a proof of concept
- Affected version/commit and environment details
- Any mitigations or suggested fixes you have identified

You can expect an acknowledgment after triage and coordinated disclosure once a fix is available.
